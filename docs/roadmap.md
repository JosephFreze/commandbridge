# CommandBridge Roadmap

## Phase 1 – Foundation
- Project structure and documentation
- Local GUI for command input
- Logging system
- Orchestrator pipeline (stubbed)

---

## Phase 2 – AI Reasoning
- AI-based safety evaluation
- AI-generated execution plans
- Plan display in the UI
- No real system execution yet

---

## Phase 3 – Local Execution
- Mouse and keyboard control
- Launching local applications
- Browser automation
- Basic OS actions (Notepad, Recycle Bin)

---

## Phase 4 – Expanded Capabilities
- File management
- Email interaction
- Short-term memory and context
- Improved error handling and recovery

---

## Phase 5 – Refinement
- Improved UI feedback
- Configurable safety modes
- Performance tuning
- Documentation cleanup

---

## Long-Term Ideas
- Plugin system for new capabilities
- Optional local AI models
- User profiles and permission presets
